# FSq-Poplar.github.io

## Live

[fsq-poplar.com](https://fsq-poplar.com/)

## Workflows

- Linted with [Super-Linter](https://github.com/github/super-linter) on push
- If no linting issues, deployed to GitHub pages on push to master

## Usage

Dev:

```bash
npm install
npm start
```

Lint:

```bash
./styles
```

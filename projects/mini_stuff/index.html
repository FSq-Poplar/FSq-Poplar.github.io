Binder Sleeper Laptop

No GitHub repo for this one; an old netbook of mine had its case crack, to the point where it wasn't really portable anymore unless I wanted to break the thing in half. Fortunately, it was still functional as a computer and I ended up taking the thing out of its chassis.

I remembered seeing pentesting kits built into makeup kits, and had the idea to build the laptop into a binder I had lying around. After buying a bluetooth keyboard/trackpad, it only took a few hours to put together this sleeper laptop.

Storage Visualizer

This was actually made from a school assignment in which we made a storage visualizer to practice working with trees. I needed to find ogg music files in a game folder, and ended up adding the file type colorization to the assignment to find their directory; later I rebuilt it from scratch and this tool took shape.
